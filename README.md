# Claude Code Skills Marketplace

A marketplace repository for reusable Claude Code skills.

This repository provides production-ready workflows, automation skills, and reusable agent capabilities for Claude Code.

---

## Features

- Claude Code marketplace compatible
- Reusable workflow skills
- Slash-command friendly
- Modular skill architecture
- Tool permission isolation
- CI/CD friendly repository structure
- Examples and templates support

---

## Installation

Install directly from the Claude Code marketplace:

```bash
claude marketplace install lshgdut/cc-marketplace
```

The cc marketplace provides some other related plugins for Claude Code.

- Register the marketplace:

  ```bash
  /plugin marketplace add lshgdut/cc-marketplace
  ```

After installation, Claude Code automatically discovers and loads the skills.

---

## Usage

Invoke skills naturally in Claude Code.

Examples:

```text
/glab-merge-request
```

```text
Create a GitLab merge request
```

```text
Generate release notes for the current branch
```
